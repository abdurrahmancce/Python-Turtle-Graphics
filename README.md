🎨 Turtle Colorful Spiral Pattern

This is a simple Python graphics project that creates a beautiful colorful spiral pattern using the Turtle Graphics library.
The program draws lines repeatedly while changing colors to produce an artistic star-like spiral on a black background.

This project is beginner-friendly and a great way to learn how loops, colors, and graphics work in Python.

📸 Output

The program generates a colorful geometric pattern that looks like a spiral star with multiple colors on a black screen.
